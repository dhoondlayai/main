# dhoondlayai Site

This is the repository for the dhoondlayai organization landing page, built with [VitePress](https://vitepress.dev/).

## Project Structure

- `docs/`: Markdown files for the site content.
  - `.vitepress/`: VitePress configuration and theme customization.
  - `public/`: Static assets like images and icons.
  - `index.md`: The landing page content.

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm

### Installation

```bash
npm install
```

### Development

Run the development server:

```bash
npm run docs:dev
```

### Build

Build the static site for production:

```bash
npm run docs:build
```

### Preview

Preview the production build:

```bash
npm run docs:preview
```
